### DNS(Domain Name System)
도메인 이름이 IP주소와 매핑되어 있어, 이름만으로 최종 서버 ip에 접근할 수 있게 해준다. (마치 전화번호부와 같은 기능)
DNS 서버는 이름을 IP 주소로 변환하여 도메인 이름을 웹 브라우저에 입력할 때, 최종 사용자를 어떤 서버에 연결할 것인지를 제어한다. 
이러한 요청을 '쿼리'라 한다.
